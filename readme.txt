== Description ==

The wp-weblink plugin let's you install a polish link-exchange system e-weblink.com in your blog footer section.


== Installation ==

1. Open 'wp-weblink.php'and replace "weblink_file.php" with your unique e-weblink.com file.
2. Upload `wp-weblink.php` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Enjoy

== Frequently Asked Questions ==

= The plugin doesn't seem to work with my theme. =

You will need to edit the "wp-content/themes/your-theme/footer.php" and paste <code><?php wp_footer();></code>.
